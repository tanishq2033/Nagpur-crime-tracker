
# Nagpur Crime Heatmap

Nagpur Crime Heatmap is a full-stack web application designed to visualize crime-prone areas across Nagpur city using an interactive heatmap dashboard. The platform provides a modern dark-themed intelligence-style interface that maps crime intensity based on location data, helping users identify hotspot zones and analyze patterns across different areas and time periods.

The application uses a React-based frontend with an interactive map interface and a Python FastAPI backend to serve crime data through REST APIs. Crime incidents are displayed as glowing heat zones on a city map, allowing users to quickly identify high-risk locations.

## Key Features

* Interactive city heatmap centered on Nagpur
* Real-time hotspot visualization
* Dark intelligence dashboard UI
* Crime category filters
* Day / Night / All time filters
* Incident statistics panel
* Backend API integration using FastAPI
* Offline fallback dataset support

## Tech Stack

* **Frontend:** React, Tailwind CSS, Leaflet.js
* **Backend:** FastAPI (Python)
* **Data:** JSON-based crime dataset
* **Visualization:** Heatmap overlays with custom intensity gradients

## Objective

The goal of this project is to transform raw location-based crime data into a meaningful visual dashboard that helps in identifying dangerous zones, understanding crime concentration, and improving situational awareness.

This project demonstrates full-stack development, API integration, geospatial visualization, and modern UI/UX design.
